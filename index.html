Skin tone shade extractor

A python project designed to categorize skin tone shades to a specific color. The project uses haar cascade to detect faces and K’ Means Clustering algorithm to detect skin tone and minimum distance to allot skin tone shade. The algorithm is pretty fast and can allot shades with good enough accuracy.
Technologies Used: Python 3, Sklearn, Numpy, OpenCV, Matplotlib.
